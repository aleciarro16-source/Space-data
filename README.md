# 🚀 SpaceX Data Science Project

## 📌 Descripción del proyecto

Este proyecto tiene como objetivo analizar datos de lanzamientos espaciales de SpaceX para entender los factores que influyen en el éxito del aterrizaje de la primera etapa de los cohetes Falcon 9.

A partir de análisis de datos, visualizaciones interactivas y modelos de machine learning, se busca predecir si un lanzamiento tendrá éxito o no.

---

## 🎯 Objetivo

- Analizar datos históricos de lanzamientos de SpaceX
- Identificar patrones de éxito y fallo
- Crear visualizaciones interactivas (Dash y Folium)
- Construir un modelo de predicción de aterrizaje de la primera etapa

---

## 🧠 Contexto

SpaceX ha revolucionado la industria espacial gracias a la reutilización de cohetes, lo que reduce significativamente los costos de lanzamiento.

Predecir si la primera etapa aterrizará con éxito permite estimar el costo de cada misión y la eficiencia de la empresa.

## 📈 Resultados

Se lograron identificar patrones clave en los lanzamientos de SpaceX, incluyendo:

- Identificación de sitios con mayor tasa de éxito
- Relación entre carga útil y éxito del lanzamiento
- Visualización de ubicaciones estratégicas
- Modelo predictivo para estimar el éxito del aterrizaje
